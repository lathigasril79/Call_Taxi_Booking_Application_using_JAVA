# Taxi Booking System

A Java-based taxi booking application that manages taxi allocations, bookings, and earnings tracking.

## Features

- Multi-Taxi Management: Create and manage multiple taxis
- Smart Taxi Allocation: Automatic taxi assignment based on proximity and earnings
- Trip Tracking: Complete booking history with detailed records
- Earnings Calculation: Real-time earnings tracking per taxi
- Input Validation: Robust exception handling for invalid inputs
- Menu-Driven Interface: User-friendly console interface
- Real-time Availability: Checks taxi availability based on time and location

## Technologies

- Java (Core Java, OOP Concepts)
- Collections Framework (ArrayList, List)
- Exception Handling (Custom Exceptions)
- I/O Streams (BufferedReader)
